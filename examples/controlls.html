jMol - a jQuery plugin for Jmol

This plugin is intended to replace the old-school Jmol.js and add the ease of use of jQuery coding style. Also this plugin is inteded to work with W3C standards compatible tags (maybe breaking backwards compatibility with old-school browsers) and get rid of improper use of inline scripting.

You are viewing this page with a JavaScript disabled browser. Please enable JavaScript to view Jmol.

This example illustrates controll scripts for Jmol viewer applet through jQuery plugin. All the scripting code is stored in HTML5 data attribute in each of the links. Also there is an echo log, that receives messages through echo callback and a hover and a pick event hadler.
